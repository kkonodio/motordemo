# motordemo
# 步进电机 PID 闭环调速

基于 STC89C52 的步进电机速度闭环控制系统，采用编码器反馈 + PID 算法，实现 10.00 ~ 20.00 rpm 范围内稳定调速。
！[原理图](./原理图.png)
---

## 硬件平台

- 主控：STC89C52
- 显示：LCD1602
- 输入：2 个独立按键
- 驱动：四相步进电机 28BYJ-48
- 反馈：编码器  欧姆龙E6B2-CWZ6C

---

## 功能

- 启停控制（按键 1 ）
- 目标转速设置（按键 2 ）
- PID 速度闭环控制
- 1602 实时显示目标转速 / 实际转速
